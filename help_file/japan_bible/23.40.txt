23 40-1 慰めよ、わたしの民を慰めよと　あなたたちの神は言われる。

23 40-2 エルサレムの心に語りかけ　彼女に呼びかけよ　苦役の時は今や満ち、彼女の咎は償われた、と。　罪のすべてに倍する報いを　主の御手から受けた、と。

23 40-3 呼びかける声がある。　主のために、荒れ野に道を備え　わたしたちの神のために、荒れ地に広い道を通せ。

23 40-4 谷はすべて身を起こし、山と丘は身を低くせよ。　険しい道は平らに、狭い道は広い谷となれ。

23 40-5 主の栄光がこうして現れるのを　肉なる者は共に見る。　主の口がこう宣言される。

23 40-6 呼びかけよ、と声は言う。　わたしは言う、何と呼びかけたらよいのか、と。　肉なる者は皆、草に等しい。　永らえても、すべては野の花のようなもの。

23 40-7 草は枯れ、花はしぼむ。　主の風が吹きつけたのだ。　この民は草に等しい。

23 40-8 草は枯れ、花はしぼむが　わたしたちの神の言葉はとこしえに立つ。

23 40-9 高い山に登れ　良い知らせをシオンに伝える者よ。　力を振るって声をあげよ　良い知らせをエルサレムに伝える者よ。　声をあげよ、恐れるな　ユダの町々に告げよ。　見よ、あなたたちの神

23 40-10 見よ、主なる神。　彼は力を帯びて来られ　御腕をもって統治される。　見よ、主のかち得られたものは御もとに従い　主の働きの実りは御前を進む。

23 40-11 主は羊飼いとして群れを養い、御腕をもって集め　小羊をふところに抱き、その母を導いて行かれる。

23 40-12 手のひらにすくって海を量り　手の幅をもって天を測る者があろうか。　地の塵を升で量り尽くし　山々を秤にかけ　丘を天秤にかける者があろうか。

23 40-13 主の霊を測りうる者があろうか。　主の企てを知らされる者があろうか。

23 40-14 主に助言し、理解させ、裁きの道を教え　知識を与え、英知の道を知らせうる者があろうか。

23 40-15 見よ、国々は革袋からこぼれる一滴のしずく　天秤の上の塵と見なされる。　島々は埃ほどの重さも持ちえない。

23 40-16 レバノンの森も薪に足りず　その獣もいけにえに値しない。

23 40-17 主の御前に、国々はすべて無に等しく　むなしくうつろなものと見なされる。

23 40-18 お前たちは、神を誰に似せ　どのような像に仕立てようというのか。

23 40-19 職人は偶像を鋳て造り　金箔を作ってかぶせ、銀の鎖を付ける。

23 40-20 献げ物にする桑の木、えり抜きの朽ちない木を　巧みな職人は捜し出し、像を造り、据え付ける。

23 40-21 お前たちは知ろうとせず聞こうとしないのか　初めから告げられてはいなかったのか　理解していなかったのか、地の基の置かれた様を。

23 40-22 主は地を覆う大空の上にある御座に着かれる。　地に住む者は虫けらに等しい。　主は天をベールのように広げ、天幕のように張り　その上に御座を置かれる。

23 40-23 主は諸侯を無に等しいものとし　地を治める者をうつろなものとされる。

23 40-24 彼らは植えられる間もなく、種蒔かれる間もなく　地に根を張る間もなく　風が吹きつけてこれを枯らす。　嵐がわらのように巻き上げる。

23 40-25 お前たちはわたしを誰に似せ　誰に比べようとするのか、と聖なる神は言われる。

23 40-26 目を高く上げ、誰が天の万象を創造したかを見よ。　それらを数えて、引き出された方　それぞれの名を呼ばれる方の　力の強さ、激しい勢いから逃れうるものはない。

23 40-27 ヤコブよ、なぜ言うのか　イスラエルよ、なぜ断言するのか　わたしの道は主に隠されている、と　わたしの裁きは神に忘れられた、と。

23 40-28 あなたは知らないのか、聞いたことはないのか。　主は、とこしえにいます神　地の果てに及ぶすべてのものの造り主。　倦むことなく、疲れることなく　その英知は究めがたい。

23 40-29 疲れた者に力を与え　勢いを失っている者に大きな力を与えられる。

23 40-30 若者も倦み、疲れ、勇士もつまずき倒れようが

23 40-31 主に望みをおく人は新たな力を得　鷲のように翼を張って上る。　走っても弱ることなく、歩いても疲れない。
