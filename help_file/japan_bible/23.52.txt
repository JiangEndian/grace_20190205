23 52-1 奮い立て、奮い立て　力をまとえ、シオンよ。　輝く衣をまとえ、聖なる都、エルサレムよ。　無割礼の汚れた者が　あなたの中に攻め込むことは再び起こらない。

23 52-2 立ち上がって塵を払え、捕らわれのエルサレム。　首の縄目を解け、捕らわれの娘シオンよ。

23 52-3 主はこう言われる。　「ただ同然で売られたあなたたちは　銀によらずに買い戻される」と。

23 52-4 主なる神はこう言われる。初め、わたしの民はエジプトに下り、そこに宿った。また、アッシリア人は故なくこの民を搾取した。

23 52-5 そして今、ここで起こっていることは何か、と主は言われる。わたしの民はただ同然で奪い去られ、支配者たちはわめき、わたしの名は常に、そして絶え間なく侮られている、と主は言われる。

23 52-6 それゆえ、わたしの民はわたしの名を知るであろう。それゆえその日には、わたしが神であることを、「見よ、ここにいる」と言う者であることを知るようになる。

23 52-7 いかに美しいことか　山々を行き巡り、良い知らせを伝える者の足は。　彼は平和を告げ、恵みの良い知らせを伝え　救いを告げ　あなたの神は王となられた、と　シオンに向かって呼ばわる。

23 52-8 その声に、あなたの見張りは声をあげ　皆共に、喜び歌う。　彼らは目の当たりに見る　主がシオンに帰られるのを。

23 52-9 歓声をあげ、共に喜び歌え、エルサレムの廃虚よ。　主はその民を慰め、エルサレムを贖われた。

23 52-10 主は聖なる御腕の力を　国々の民の目にあらわにされた。　地の果てまで、すべての人が　わたしたちの神の救いを仰ぐ。

23 52-11 立ち去れ、立ち去れ、そこを出よ　汚れたものに触れるな。　その中から出て、身を清めよ　主の祭具を担う者よ。

23 52-12 しかし、急いで出る必要はない　逃げ去ることもない。　あなたたちの先を進むのは主であり　しんがりを守るのもイスラエルの神だから。

23 52-13 見よ、わたしの僕は栄える。　はるかに高く上げられ、あがめられる。

23 52-14 かつて多くの人をおののかせたあなたの姿のように　彼の姿は損なわれ、人とは見えず　もはや人の子の面影はない。

23 52-15 それほどに、彼は多くの民を驚かせる。　彼を見て、王たちも口を閉ざす。　だれも物語らなかったことを見　一度も聞かされなかったことを悟ったからだ。
