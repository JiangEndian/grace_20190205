28 14-1 サマリアは罰せられる。　その神に背いたからだ。　住民は剣に倒れ　幼子は打ち殺され　妊婦は引き裂かれる。

28 14-2 イスラエルよ、立ち帰れ　あなたの神、主のもとへ。　あなたは咎につまずき、悪の中にいる。

28 14-3 誓いの言葉を携え　主に立ち帰って言え。　「すべての悪を取り去り　恵みをお与えください。　この唇をもって誓ったことを果たします。

28 14-4 アッシリアはわたしたちの救いではありません。　わたしたちはもはや軍馬に乗りません。　自分の手が造ったものを　再びわたしたちの神とは呼びません。　親を失った者は　あなたにこそ憐れみを見いだします。」

28 14-5 わたしは背く彼らをいやし　喜んで彼らを愛する。　まことに、わたしの怒りは彼らを離れ去った。

28 14-6 露のようにわたしはイスラエルに臨み　彼はゆりのように花咲き　レバノンの杉のように根を張る。

28 14-7 その若枝は広がり　オリーブのように美しく　レバノンの杉のように香る。

28 14-8 その陰に宿る人々は再び　麦のように育ち　ぶどうのように花咲く。　彼はレバノンのぶどう酒のようにたたえられる。

28 14-9 ああエフライム　なおも、わたしを偶像と比べるのか。　彼の求めにこたえ　彼を見守るのはわたしではないか。　わたしは命に満ちた糸杉。　あなたは、わたしによって実を結ぶ。

28 14-10 知恵ある者はこれらのことをわきまえよ。　わきまえある者はそれを悟れ。　主の道は正しい。　神に従う者はその道に歩み　神に背く者はその道につまずく。
