23 26-1 その日には、ユダの地でこの歌がうたわれる。　我らには、堅固な都がある。　救いのために、城壁と堡塁が築かれた。

23 26-2 城門を開け　神に従い、信仰を守る民が入れるように。

23 26-3 堅固な思いを、あなたは平和に守られる　あなたに信頼するゆえに、平和に。

23 26-4 どこまでも主に信頼せよ、主こそはとこしえの岩。

23 26-5 主は高い所に住まう者を引きおろし　築き上げられた都を打ち倒し　地に打ち倒して、塵に伏させる。

23 26-6 貧しい者の足がそれを踏みにじり　弱い者の足が踏みつけて行く。

23 26-7 神に従う者の行く道は平らです。　あなたは神に従う者の道をまっすぐにされる。

23 26-8 主よ、あなたの裁きによって定められた道を歩み　わたしたちはあなたを待ち望みます。　あなたの御名を呼び、たたえることは　わたしたちの魂の願いです。

23 26-9 わたしの魂は夜あなたを捜し　わたしの中で霊はあなたを捜し求めます。　あなたの裁きが地に行われるとき　世界に住む人々は正しさを学ぶでしょう。

23 26-10 神に逆らう者は、憐れみを受けても　正しさを学ぶことがありません。公正の行われている国で不正を行い　主の威光を顧みようとしません。

23 26-11 主よ、あなたの高く上げられた御手を　彼らは仰ごうとしません。　民に対するあなたの熱情を仰がせ　彼らに恥を受けさせてください。　敵対する者に向けられるあなたの火が　彼らを焼き尽くしますように。

23 26-12 主よ、平和をわたしたちにお授けください。　わたしたちのすべての業を　成し遂げてくださるのはあなたです。

23 26-13 わたしたちの神なる主よ　あなた以外の支配者が我らを支配しています。　しかしわたしたちは　あなたの御名だけを唱えます。

23 26-14 死者が再び生きることはなく　死霊が再び立ち上がることはありません。　それゆえ、あなたは逆らう者を罰し、滅ぼし　彼らの記憶をすべて無に帰されました。

23 26-15 主よ、あなたはその民を増やされました。　あなたはその民を増やし、栄光を示し　その土地の境を四方に広げられました。

23 26-16 主よ、苦難に襲われると　人々はあなたを求めます。　あなたの懲らしめが彼らに臨むと　彼らはまじないを唱えます。

23 26-17 妊婦に出産のときが近づくと　もだえ苦しみ、叫びます。　主よ、わたしたちもあなたの御前で　このようでした。

23 26-18 わたしたちははらみ、産みの苦しみをしました。　しかしそれは風を産むようなものでした。　救いを国にもたらすこともできず　地上に住む者を　産み出すこともできませんでした。

23 26-19 あなたの死者が命を得　わたしのしかばねが立ち上がりますように。　塵の中に住まう者よ、目を覚ませ、喜び歌え。　あなたの送られる露は光の露。　あなたは死霊の地にそれを降らせられます。

23 26-20 さあ、わが民よ、部屋に入れ。　戸を堅く閉ざせ。　しばらくの間、隠れよ　激しい憤りが過ぎ去るまで。

23 26-21 見よ、主はその御座を出て　地に住む者に、それぞれの罪を問われる。　大地はそこに流された血をあらわに示し　殺された者をもはや隠そうとはしない。
