3 14-1 主はモーセに仰せになった。

3 14-2 以下は重い皮膚病を患った人が清めを受けるときの指示である。彼が祭司のもとに連れて来られると、

3 14-3 祭司は宿営の外に出て来て、調べる。患者の重い皮膚病が治っているならば、

3 14-4 祭司は清めの儀式をするため、その人に命じて、生きている清い鳥二羽と、杉の枝、緋糸、ヒソプの枝を用意させる。

3 14-5 次に、祭司は新鮮な水を満たした土器の上で鳥の一羽を殺すように命じる。

3 14-6 それから、杉の枝、緋糸、ヒソプおよび生きているもう一羽の鳥を取り、さきに新鮮な水の上で殺された鳥の血に浸してから、

3 14-7 清めの儀式を受ける者に七度振りかけて清める。その後、この生きている鳥は野に放つ。

3 14-8 清めの儀式を受けた者は、衣服を水洗いし、体の毛を全部そって身を洗うと、清くなる。この後、彼は宿営に戻ることができる。しかし、七日間は自分の天幕の外にいなければならない。

3 14-9 彼は七日目に体の毛を全部、すなわち、頭髪、ひげ、まゆ毛、その他の毛もすべてそる。そして、衣服を水洗いし、身を洗う。こうして、彼は清くなる。

3 14-10 八日目に、彼は無傷の雄羊二匹、無傷の一歳の雌羊一匹、オリーブ油を混ぜた十分の三エファの上等の小麦粉の献げ物、一ログのオリーブ油を調える。

3 14-11 清めの儀式を執行する祭司は、その儀式にあずかる者にこれらの献げ物を持たせ、臨在の幕屋の入り口の主の御前に立たせる。

3 14-12 祭司は雄羊の一匹を取り、それを一ログのオリーブ油と共に、賠償の献げ物としてささげ、主の御前に奉納物とする。

3 14-13 次に、この雄羊を贖罪の献げ物や焼き尽くす献げ物を屠る場所、つまり聖域で屠る。この賠償の献げ物は、贖罪の献げ物と同様、祭司のものである。これは神聖なものである。

3 14-14 祭司はこの献げ物の雄羊の血を取って、清めの儀式を受ける者の右の耳たぶ、右手の親指、右足の親指に塗る。

3 14-15 祭司は、一ログのオリーブ油の一部を取って自分の左の手のひらに注ぎ、

3 14-16 そこに右手の指を浸してその油を七度主の御前に振りまく。

3 14-17 次に、手のひらに残ったオリーブ油の一部を、清めの儀式を受ける者の右の耳たぶ、右手の親指、右足の親指に塗り、更に賠償の献げ物の血の上にも塗る。

3 14-18 再び手のひらに残ったオリーブ油は清めの儀式を受ける者の頭に塗る。祭司はこのようにして、彼のために主の御前に贖いの儀式を行う。

3 14-19 祭司は、贖罪の献げ物をささげて、彼のために汚れを清める儀式を行う。最後に焼き尽くす献げ物を屠る。

3 14-20 祭司は焼き尽くす献げ物と穀物の献げ物を祭壇で燃やしてささげる。祭司がこうして、彼のために贖いの儀式を行うと、彼は清くなる。

3 14-21 もし、彼が貧しくて前記のものに手が届かないならば、自分の贖いの儀式のための奉納物として賠償の献げ物の雄羊一匹、更に穀物の献げ物のためにオリーブ油を混ぜた上等の小麦粉十分の一エファ、および一ログのオリーブ油を調える。

3 14-22 更に、手が届く二羽の山鳩か、家鳩を取り、一羽を贖罪の献げ物、他を焼き尽くす献げ物とする。

3 14-23 八日目に、彼は清めのためにこれらのものを祭司のもとに、すなわち、臨在の幕屋の入り口の主の御前に携える。

3 14-24 祭司は賠償の献げ物の雄羊と一ログのオリーブ油を取って、主の御前に奉納物とする。

3 14-25 次に、賠償の献げ物の雄羊を屠り、その血を取って清めの儀式を受ける者の右の耳たぶ、右手の親指、右足の親指に塗る。

3 14-26 祭司は自分の左の手のひらにオリーブ油を注ぎ、

3 14-27 そこに右手の指を浸してその油を七度主の御前に振りまく。

3 14-28 次に、手のひらのオリーブ油を、清めの儀式を受ける者の右の耳たぶ、右手の親指、右足の親指に塗り、更に賠償の献げ物の血の上にも塗る。

3 14-29 手のひらに残ったオリーブ油は清めの儀式を受ける者の頭に塗って、主の御前で彼のために贖いの儀式を行う。

3 14-30 最後に、手が届く山鳩か、家鳩のうちの一羽を、

3 14-31 つまり、手が届くものを一羽贖罪の献げ物とし、他を焼き尽くす献げ物とし、これに穀物の献げ物を加えてささげる。祭司はこのようにして、彼のために主の御前に贖いの儀式を行う。

3 14-32 以上は重い皮膚病を患った人が清めを受けるときに、正規の献げ物に手が届かない場合の指示である。

3 14-33 主はモーセとアロンにこう仰せになった。

3 14-34 あなたたちが所有地としてわたしから与えられるカナンの土地に入るとき、あなたたちの所有地で家屋にかびが生じるならば、

3 14-35 家の主人は祭司に「かびらしきものがわたしの家屋に生じました」と報告する。

3 14-36 祭司は、かびの状態を見に入る前に、その家屋の中にある物を屋外に出すようにと命じて、家屋の中にある物が全部汚れないようにしてから、家屋を調べるために入り、

3 14-37 かびの状態を見る。家屋の壁に青かびか、赤かびが生じており、壁の内部にまで及んでいるように見えるならば、

3 14-38 祭司は家から出て入り口に立ち、家屋を一週間封鎖する。

3 14-39 七日目に、祭司は再び来て見る。かびが家屋の壁に広がっているならば、

3 14-40 祭司は命じて、かびが生じている部分の石材を抜き取り、町の外にある汚れた場所に捨てさせ、

3 14-41 家屋の内側を削り取らせて、削り取ったしっくいを町の外にある汚れた場所に捨てさせ、

3 14-42 別の石材を取って元の石材の代わりに入れさせ、別のしっくいで家屋の内部を塗らせる。

3 14-43 しかし、石材を抜き取り、家屋の内側を削り取って、しっくいで塗った後でも、再びかびが生じて家屋に広がるならば、

3 14-44 祭司は来て調べる。かびが家屋に広がっているならば、それは悪性のかびであり、その家屋は汚れている。

3 14-45 それを打ち壊し、石材、木材、しっくいをすべて町の外にある汚れた場所に運び出させる。

3 14-46 封鎖期間中にこの家屋に入った者はすべて、夕方まで汚れる。

3 14-47 また、この家屋で寝たり、食べたりした者は衣服を水洗いする。

3 14-48 家屋をしっくいで塗り変えた後、祭司が見に来て、かびが広がっていなければ、祭司はその家屋を清いと宣言する。かびは治まったのである。

3 14-49 そして、この家屋の汚れを清めるために、二羽の鳥、杉の枝、緋糸およびヒソプを用意する。

3 14-50 まず、新鮮な水を満たした土器の上で鳥の一羽を殺す。

3 14-51 次に、杉の枝、ヒソプ、緋糸と共にもう一羽の生きている鳥を取って、さきに殺した鳥の血と新鮮な水に浸し、家屋に七度振りまく。

3 14-52 こうして、鳥の血、新鮮な水、生きた鳥、杉の枝、ヒソプおよび緋糸をもって家屋を清める。

3 14-53 その後、祭司は生きている鳥を町の外の野に放つ。こうして、家屋のための贖いの儀式を行うと、家は清められる。

3 14-54 以上は、あらゆる重い皮膚病、白癬、

3 14-55 衣服と家屋のかび、

3 14-56 湿疹、斑点、疱疹に関する、

3 14-57 汚れと清めの宣告の時についての指示である。
