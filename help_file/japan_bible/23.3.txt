23 3-1 見よ、主なる万軍の神は　支えとなり、頼みとなる者を　また、パンによる支え、水による支えをも　エルサレムとユダから取り去られる。

23 3-2 勇士と戦士、裁きを行う者と預言者　占い師と長老

23 3-3 五十人の長と尊敬される者　参議、魔術師、呪術師などを取り去られる。

23 3-4 わたしは若者を支配者にした。　気ままな者が国を治めるようになる。

23 3-5 民は隣人どうしで虐げ合う。　若者は長老に、卑しい者は尊い者に無礼を働く。

23 3-6 人は父の家で兄弟に取りすがって言う。　「お前にはまだ上着がある。　我らの指導者になり　この破滅の始末をしてくれ」と。

23 3-7 だがその日には、彼も声をあげる。　「わたしにも手当てはできない。　家にはパンもなければ上着もない。　わたしを民の指導者にしてもだめだ」と。

23 3-8 エルサレムはよろめき、ユダは倒れた。　彼らは舌と行いをもって主に敵対し　その栄光のまなざしに逆らった。

23 3-9 彼らの表情が既に証言している。　ソドムのような彼らの罪を表して、隠さない。　災いだ、彼らは悪の報いを受ける。

23 3-10 しかし言え、主に従う人は幸い、と。　彼らは自分の行いの実を食べることができる。

23 3-11 主に逆らう悪人は災いだ。　彼らはその手の業に応じて報いを受ける。

23 3-12 わたしの民は、幼子に追い使われ　女に支配されている。　わたしの民よ　お前たちを導く者は、迷わせる者で　行くべき道を乱す。

23 3-13 主は争うために構え　民を裁くために立たれる。

23 3-14 主は裁きに臨まれる　民の長老、支配者らに対して。　「お前たちはわたしのぶどう畑を食い尽くし　貧しい者から奪って家を満たした。

23 3-15 何故、お前たちはわたしの民を打ち砕き　貧しい者の顔を臼でひきつぶしたのか」と　主なる万軍の神は言われる。

23 3-16 主は言われる。　シオンの娘らは高慢で、首を伸ばして歩く。　流し目を使い、気取って小股で歩き　足首の飾りを鳴らしている。

23 3-17 主はシオンの娘らの頭をかさぶたで覆い　彼女らの額をあらわにされるであろう。

23 3-18 その日には、主は飾られた美しさを奪われる。足首の飾り、額の飾り、三日月形の飾り、

23 3-19 耳輪、腕輪、ベール、

23 3-20 頭飾り、すね飾り、飾り帯、匂袋、お守り、

23 3-21 指輪、鼻輪、

23 3-22 晴れ着、肩掛け、スカーフ、手提げ袋、

23 3-23 紗の衣、亜麻布の肌着、ターバン、ストールなどを。

23 3-24 芳香は悪臭となり、帯は縄に変わり　編んだ髪はそり落とされ　晴れ着は粗布に変わり　美しさは恥に変わる。

23 3-25 シオンの男らは剣に倒れ　勇士は戦いに倒れる。

23 3-26 シオンの城門は嘆き悲しみ　奪い尽くされて、彼女は地に座る。
