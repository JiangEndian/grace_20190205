23 64-1 柴が火に燃えれば、湯が煮えたつように　あなたの御名が敵に示されれば　国々は御前に震える。

23 64-2 期待もしなかった恐るべき業と共に降られれば　あなたの御前に山々は揺れ動く。

23 64-3 あなたを待つ者に計らってくださる方は　神よ、あなたのほかにはありません。　昔から、ほかに聞いた者も耳にした者も　目に見た者もありません。

23 64-4 喜んで正しいことを行い　あなたの道に従って、あなたを心に留める者を　あなたは迎えてくださいます。　あなたは憤られました　わたしたちが罪を犯したからです。　しかし、あなたの御業によって　わたしたちはとこしえに救われます。

23 64-5 わたしたちは皆、汚れた者となり　正しい業もすべて汚れた着物のようになった。　わたしたちは皆、枯れ葉のようになり　わたしたちの悪は風のように　わたしたちを運び去った。

23 64-6 あなたの御名を呼ぶ者はなくなり　奮い立ってあなたにすがろうとする者もない。　あなたはわたしたちから御顔を隠し　わたしたちの悪のゆえに、力を奪われた。

23 64-7 しかし、主よ、あなたは我らの父。　わたしたちは粘土、あなたは陶工　わたしたちは皆、あなたの御手の業。

23 64-8 どうか主が、激しく怒られることなく　いつまでも悪に心を留められることなく　あなたの民であるわたしたちすべてに　目を留めてくださるように。

23 64-9 あなたの聖なる町々は荒れ野となった。　シオンは荒れ野となり、エルサレムは荒廃し

23 64-10 わたしたちの輝き、わたしたちの聖所　先祖があなたを賛美した所は、火に焼かれ　わたしたちの慕うものは廃虚となった。

23 64-11 それでもなお、主よ、あなたは御自分を抑え　黙して、わたしたちを苦しめられるのですか。
