23 42-1 見よ、わたしの僕、わたしが支える者を。　わたしが選び、喜び迎える者を。　彼の上にわたしの霊は置かれ　彼は国々の裁きを導き出す。

23 42-2 彼は叫ばず、呼ばわらず、声を巷に響かせない。

23 42-3 傷ついた葦を折ることなく　暗くなってゆく灯心を消すことなく　裁きを導き出して、確かなものとする。

23 42-4 暗くなることも、傷つき果てることもない　この地に裁きを置くときまでは。　島々は彼の教えを待ち望む。

23 42-5 主である神はこう言われる。　神は天を創造して、これを広げ　地とそこに生ずるものを繰り広げ　その上に住む人々に息を与え　そこを歩く者に霊を与えられる。

23 42-6 主であるわたしは、恵みをもってあなたを呼び　あなたの手を取った。　民の契約、諸国の光として　あなたを形づくり、あなたを立てた。

23 42-7 見ることのできない目を開き　捕らわれ人をその枷から　闇に住む人をその牢獄から救い出すために。

23 42-8 わたしは主、これがわたしの名。　わたしは栄光をほかの神に渡さず　わたしの栄誉を偶像に与えることはしない。

23 42-9 見よ、初めのことは成就した。　新しいことをわたしは告げよう。　それが芽生えてくる前に　わたしはあなたたちにそれを聞かせよう。

23 42-10 新しい歌を主に向かって歌え。　地の果てから主の栄誉を歌え。　海に漕ぎ出す者、海に満ちるもの　島々とそこに住む者よ。

23 42-11 荒れ野とその町々よ。　ケダル族の宿る村々よ、呼ばわれ。　セラに住む者よ、喜び歌え。　山々の頂から叫び声をあげよ。

23 42-12 主に栄光を帰し　主の栄誉を島々に告げ知らせよ。

23 42-13 主は、勇士のように出で立ち　戦士のように熱情を奮い起こし　叫びをあげ、鬨の声をあげ、敵を圧倒される。

23 42-14 わたしは決して声を立てず　黙して、自分を抑えてきた。　今、わたしは子を産む女のようにあえぎ　激しく息を吸い、また息を吐く。

23 42-15 わたしは山も丘も廃虚とし、草をすべて枯らす。　大河を島に変え、湖を干す。

23 42-16 目の見えない人を導いて知らない道を行かせ　通ったことのない道を歩かせる。　行く手の闇を光に変え　曲がった道をまっすぐにする。　わたしはこれらのことを成就させ　見捨てることはない。

23 42-17 偶像に依り頼む者　鋳た像に向かって　あなたたちがわたしたちの神、と言う者は　甚だしく恥を受けて退く。

23 42-18 耳の聞こえない人よ、聞け。　目の見えない人よ、よく見よ。

23 42-19 わたしの僕ほど目の見えない者があろうか。　わたしが遣わす者ほど　耳の聞こえない者があろうか。　わたしが信任を与えた者ほど　目の見えない者　主の僕ほど目の見えない者があろうか。

23 42-20 多くのことが目に映っても何も見えず　耳が開いているのに、何も聞こえない。

23 42-21 主は御自分の正しさゆえに　教えを偉大なものとし、輝かすことを喜ばれる。

23 42-22 この民は略奪され、奪われ　皆、穴の中に捕らえられ、牢につながれている。　略奪に遭っても、助け出す者はなく　奪われても、返せと言う者はない。

23 42-23 あなたたちの中にこれを聞き取る者があるか。　後の日のために注意して聞く者があるか。

23 42-24 奪う者にヤコブを渡し　略奪する者にイスラエルを渡したのは誰か。　それは主ではないか　この方にわたしたちも罪を犯した。　彼らは主の道に歩もうとせず　その教えに聞き従おうとしなかった。

23 42-25 主は燃える怒りを注ぎ出し　激しい戦いを挑まれた。　その炎に囲まれても、悟る者はなく　火が自分に燃え移っても、気づく者はなかった。
